Project Iris

:: Definitions

  • modules

    Any item that can be docked into the bar. Modules open panels

  • User module

    Left module - static width container housing user functionality : login, register etc...

  • Sharing module

    Right module - static width container that houses communication functionality : rss, email , print

  • Publication module

    Center module - collection area which publications are stored to be shared

  • Pubs

    Short for publications - articles that are stored in the publication module

  • panels

    window that open in reaction to a module interaction

  • Timeline ?

    Shareable user history
